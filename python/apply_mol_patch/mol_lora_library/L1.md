# L1 living/vita/tau3

## Description
Specialist LoRA for personal-agent and customer-service workflows. The slash-separated task label means one adapter covers LivingBench real-world coordination, VitaBench Chinese service-tool workflows, and Tau3 English policy-bound customer-service workflows.

## Routing Rules
- Choose L1 for concrete real-world coordination involving people, time, budgets, travel, health, family, school, housing, restaurants, errands, appointments, messages, meetings, purchases, repairs, or practical plans.
- Choose L1 for VitaBench-style Chinese consumer-service workflows: OTA travel, trains, flights, hotels, attractions, food delivery, nearby search, store recommendations, orders, cancellation, modification, payment, weather, or maps.
- Choose L1 for Tau3-style English customer-service workflows in airline, banking, retail, or telecom domains, especially with policy text, account verification, reservations, credit cards, retail orders, telecom lines, or structured IDs.
- Prefer L1 over L0 when the user asks the assistant to handle a concrete personal, service, booking, order, policy, or customer-support workflow.
- Do not choose L1 for repository repair, shell/terminal tasks, local file artifact pipelines, or UI/app screen generation.
