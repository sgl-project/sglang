# LoRA Support

This page lists the LoRAs that are currently supported for each base model in the **SGLang Diffusion** pipeline.

This registry is curated and intentionally non-exhaustive. If a LoRA is not listed here, it may still work with additional conversion, remapping, or custom handling, but such cases are outside the scope of this document.
