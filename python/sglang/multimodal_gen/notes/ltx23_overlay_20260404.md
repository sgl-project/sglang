    - 缺少 2.3 aggregate embed 前的 rescale
- 已落地修复：
  - `configs/pipeline_configs/ltx_2.py`
    - 当 `caption_proj_before_connector=True` 时，`_gemma_postprocess_func` 改走 2.3 的 per-token RMS norm
  - `runtime/models/adapter/ltx_2_connector.py`
    - 在 `video_aggregate_embed` / `audio_aggregate_embed` 前补官方一致的 rescale
  - 补了最小单测锁住 2.3 prompt pre-processing 逻辑
- 修复后重新 compare 的结果：
  - `video_features` cosine:
    - prompt: `0.9999982`
    - negative prompt: `0.9999956`
  - `audio_features` cosine:
    - prompt: `0.9999979`
    - negative prompt: `0.9999948`
  - 含义：
    - `feature_extractor` 级别已经基本对齐
    - 剩余 prompt-stage 漂移主要落在 `connector transformer` 自身，而不是 Gemma hidden-state postprocess
