## Command

```
python python/sglang/srt/layers/attention/nsa/hisa/benchmark/benchmark_indexer.py
# defaults: n_head=64, head_dim=128, topk_tokens=2048,
#           k_block_size=128, block_topk=64,
#           paged_block_size=64, max_model_len=131072, num_sms=132,
#           num_warmups=5, num_iters=20
```

Baseline = `deep_gemm.fp8_{mqa,paged_mqa}_logits` + `torch.topk`.
Hierarchy = `fp8_native_hierarchy_{,paged_}mqa_logits` + `torch.topk` + gather.

Units below: median ms ± stdev ms over 20 iters (5 warmup, L2 flushed per iter).

## Notes

- GPU was exclusive (`nvidia-smi` showed 143 GB free before run).
- 20 iters + L2 flush per iter; stdev is small except prefill 64K (7.15 ms, ~6%).
- No vLLM-env baseline to compare against — previous benchmark numbers were not saved.
- Kernels changed since vLLM patch (at minimum block_mqa → pool_mqa rename); sweep direction here is not purely "same kernels, different framework".
