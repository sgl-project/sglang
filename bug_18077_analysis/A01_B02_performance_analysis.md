## 当前已知数据

### 初步基准测试结果（来自 Issue 评论）

#### 测试配置
- **模型**: Wan-AI/Wan2.1-T2V-1.3B-Diffusers
- **任务**: text-to-video
- **数据集**: vbench
- **并发数**: 1
- **请求数**: 10

#### Diffusers Backend 结果
| 指标 | 值 |
|------|-----|
| 吞吐量 (req/s) | 0.50 |
| 平均延迟 (s) | 2.0054 |
| 中位数延迟 (s) | 2.0053 |
| P99 延迟 (s) | 2.0067 |
| 峰值内存 (MB) | 14026.38 |

#### SGLang Backend 结果
| 指标 | 值 |
|------|-----|
| 吞吐量 (req/s) | 0.11 |
| 平均延迟 (s) | 9.2222 |
| 中位数延迟 (s) | 9.0214 |
| P99 延迟 (s) | 10.8502 |
| 峰值内存 (MB) | 8170.82 |

### 性能对比

| 指标 | Diffusers | SGLang | 差距 | 倍数 |
|------|-----------|--------|------|------|
| 吞吐量 (req/s) | 0.50 | 0.11 | -0.39 | 4.5x |
| 平均延迟 (s) | 2.01 | 9.22 | +7.21 | 4.6x |
| 峰值内存 (MB) | 14026 | 8170 | -5856 | 0.58x |

### 关键发现

1. **性能劣势**: SGLang 在延迟和吞吐量上明显落后于 Diffusers
2. **内存优势**: SGLang 使用更少的内存（约 58%）
3. **稳定性**: 两种后端的延迟分布都相对稳定（P99 接近平均值）
