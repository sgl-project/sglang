# A01_B07: 模型大小和显存需求分析

## 📊 Wan-AI/Wan2.1-T2V-1.3B-Diffusers 模型信息

### 模型规格
- **模型名称**: Wan-AI/Wan2.1-T2V-1.3B-Diffusers
- **参数量**: 1.3B (13亿参数)
- **架构**: DiT (Diffusion Transformer) + T5 文本编码器
- **任务**: Text-to-Video (T2V)

### 显存需求分析

#### 实际测试数据（来自 bug_17671_analysis）

**测试环境**:
- GPU: RTX 4090 (24GB 显存)
- 系统: WSL2
- 测试模型: Wan 2.1 1.3B

**测试结果**:
```
显存占用: > 23 GB (几乎占满 24GB)
RAM 占用: 在 WSL2 环境下会触发 OOM (Exit Code -9)
```

#### 关键发现

1. **显存占用接近 24GB**
   - 模型加载后显存几乎被吃满（> 23 GB）
   - 在 RTX 4090 上可以运行，但显存非常紧张
   - **结论**: 模型显存需求远超过 10GB，接近 24GB

2. **内存压力**
   - T5 风格的文本编码器体积巨大
   - 在 WSL2 环境下会触发系统级 OOM
   - 需要足够的系统 RAM（建议 > 32GB）

3. **性能问题**
   - 当显存占满时，会触发 GPU ↔ CPU 之间的 PCIe 内存交换
   - 单步推理耗时可能飙升至 1500 秒级别
   - 调度器可能因长时间无响应触发超时

## 🎯 RTX 4090 运行可行性

### ✅ 可以运行，但有条件

**优势**:
- RTX 4090 有 24GB 显存，刚好可以容纳模型
- 模型参数量相对较小（1.3B）

**挑战**:
- 显存占用接近上限（> 23GB），几乎没有余量
- 无法支持大 batch size
- 高分辨率生成可能受限
- 在 WSL2 环境下需要足够的系统 RAM

### 建议配置

#### 最低配置
- **GPU**: RTX 4090 (24GB) ✅
- **系统 RAM**: 至少 32GB（推荐 64GB）
- **操作系统**: Linux 原生（避免 WSL2 内存限制）

#### 推荐配置
- **GPU**: RTX 4090 (24GB) 或更高
- **系统 RAM**: 64GB+
- **操作系统**: Linux 原生
- **优化选项**:
  - 使用量化版本（如果有）
  - 启用 CPU offload（如果支持）
  - 使用较小的 batch size

## 📈 与其他模型对比

| 模型名称 | 架构类型 | 显存占用 | 系统 RAM 压力 | 运行表现 |
|---------|---------|---------|--------------|---------|
| **Wan 2.1 1.3B** | **DiT + T5** | **> 23 GB** | **极高** | **显存紧张，可能触发交换** |
| Z-Image-Turbo | DiT | 接近 24GB | 中等 | 极慢，可能超时 |

### 关键观察

**DiT / T5 型 Diffusion 模型特点**:
- 对内存/显存极其敏感
- 在默认 WSL2 环境下"几乎注定失败"
- 需要大量系统 RAM 支持 T5 编码器

## 💡 优化建议

### 1. 系统级优化
- ✅ 使用 Linux 原生系统（避免 WSL2）
- ✅ 增加系统 RAM（至少 32GB，推荐 64GB）
- ✅ 关闭不必要的后台程序

### 2. 模型级优化
- 🔍 查找量化版本（INT4/INT8）
- 🔍 使用 CPU offload（如果支持）
- 🔍 使用更小的 batch size

### 3. SGLang 配置优化
```bash
# 使用较小的 batch size
--max-num-batched-tokens 4096

# 启用内存优化
--mem-fraction-static 0.9

# 使用 CPU offload（如果支持）
--enable-offload
```

## 📝 总结

### 关于您的问题

**Q: 这个模型大吗？**
- ✅ **是的，非常大** - 显存需求接近 24GB

**Q: 4090 能跑吗？**
- ✅ **可以运行**，但显存非常紧张，几乎没有余量

**Q: 低于 10GB 吗？**
- ❌ **不是**，实际显存需求 > 23GB，远超过 10GB

### 实际建议

1. **如果只有 RTX 4090**:
   - 可以尝试运行，但需要足够的系统 RAM（32GB+）
   - 使用 Linux 原生系统（避免 WSL2）
   - 使用较小的 batch size 和分辨率
   - 做好性能较慢的心理准备

2. **如果追求更好的体验**:
   - 考虑使用更大的 GPU（如 A100 80GB）
   - 或者寻找量化版本的模型
   - 或者使用其他更轻量的模型（如 SD-Turbo）
