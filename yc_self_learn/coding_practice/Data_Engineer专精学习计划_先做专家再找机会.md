# Data Engineer 专精学习计划：先做专家，再找机会

## 🎯 核心策略：专注 Data，AI Infra 作为兴趣

### 你的决定

```
✅ 主攻方向：Data Engineer（成为专家）
✅ 兴趣方向：AI Infra / LLM（保持学习，不占用主要时间）
```

## 🎯 Data Engineer 核心技能清单

### 优先级 1：必会技能（80% 的工作）

#### 1. SQL 四大支柱 ⭐⭐⭐⭐⭐

```
组件 A：排名与 Top N
  ✅ ROW_NUMBER()、RANK()、DENSE_RANK()
  ✅ 窗口函数（PARTITION BY、ORDER BY）
  ✅ Top N 问题（LIMIT、窗口函数）
  
  练习：每天 1-2 道题
  目标：看到题目就能想到用哪个函数

组件 B：连续性问题
  ✅ LAG()、LEAD()（偏移函数）
  ✅ 自连接（找出连续记录）
  ✅ 日期连续性判断
  
  练习：每天 1-2 道题
  目标：能快速识别连续性问题

组件 C：时间维度转换
  ✅ DATE_TRUNC()、EXTRACT()
  ✅ 时间窗口（时间聚合）
  ✅ 日期计算（DATEDIFF、DATE_ADD）
  
  练习：每天 1-2 道题
  目标：熟练处理时间相关查询

组件 D：条件聚合
  ✅ SUM(CASE WHEN)、COUNT(CASE WHEN)
  ✅ GROUP BY + HAVING
  ✅ 多维度聚合
  
  练习：每天 1-2 道题
  目标：能快速写条件聚合
```

**学习计划**：
- 第 1 周：每天 2-3 道 SQL 题，覆盖四大支柱
- 第 2 周：每天 3-4 道 SQL 题，加深理解
- 第 3 周：每天 4-5 道 SQL 题，提高速度
- 第 4 周：综合练习，模拟面试

#### 2. Pandas 常用操作 ⭐⭐⭐⭐⭐

```
核心操作：
  ✅ groupby().agg()（分组聚合）
  ✅ merge()（数据关联）
  ✅ pivot_table()（数据透视）
  ✅ apply()（自定义函数）
  
数据清洗：
  ✅ drop_duplicates()（去重）
  ✅ fillna()、dropna()（处理缺失值）
  ✅ 数据筛选（条件筛选）
  
数据转换：
  ✅ 数据类型转换
  ✅ 时间格式转换
  ✅ 字符串处理

练习计划：
  - 每天 1-2 个 Pandas 操作练习
  - 每周完成一个完整的 ETL 流程
```

#### 3. ETL 开发 ⭐⭐⭐⭐

```
Extract（提取）：
  ✅ 从数据库提取（SQLAlchemy）
  ✅ 从 API 提取（requests）
  ✅ 从文件提取（CSV、JSON、Parquet）

Transform（转换）：
  ✅ 数据清洗（去重、处理缺失值）
  ✅ 数据格式转换
  ✅ 数据聚合和关联

Load（加载）：
  ✅ 存入数据仓库（to_sql）
  ✅ 存入文件（to_parquet、to_csv）
  ✅ 存入数据湖（S3）

练习计划：
  - 每周完成一个完整的 ETL 项目
  - 包含错误处理、监控、日志
```

#### 4. 数据管道设计 ⭐⭐⭐⭐

```
调度工具：
  ✅ Airflow（最常用）
  ✅ 理解 DAG、Task、依赖关系

数据建模：
  ✅ 星型模型（事实表 + 维度表）
  ✅ 雪花模型
  ✅ 维度建模

性能优化：
  ✅ SQL 优化（索引、分区）
  ✅ 查询优化（减少 JOIN、使用物化视图）
  ✅ 存储优化（分区、压缩）

练习计划：
  - 每周设计一个数据管道
  - 包含架构设计、性能优化方案
```
