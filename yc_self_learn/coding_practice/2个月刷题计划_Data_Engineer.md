# 2 个月刷题计划 - Data Engineer 求职准备

## 📋 计划概览（全面版：SQL + Python + AI Infra + 系统设计）

**时间范围**：2 个月（8 周）  
**目标岗位**：Data Engineer（主攻方向）+ AI Infra（兴趣方向）  
**你的现状**：
- ✅ Easy 题目：做出来的概率很高（尤其是 SQL Easy）
- ⚠️ SQL Medium：能写出结构，但语法一般弄不完
- ❌ LeetCode Python Medium：一般不行
- ❌ Pandas Medium：不太行

**总体策略**：
- **SQL Medium**（核心，60% 时间）：语法强化，从 50% → 90%+
- **Python Easy**（基础，15% 时间）：保持基础，偶尔练习
- **AI Infra**（兴趣，15% 时间）：SGLang 研究，系统设计准备
- **系统设计**（重要，10% 时间）：Data Engineer + AI Infra 系统设计

**时间分配**：
- 每天晚上 2-3 小时
- SQL Medium：60 分钟（核心）
- Python Easy：20-30 分钟（基础）
- AI Infra / 系统设计：30-40 分钟（兴趣 + 准备）
- 总结：10-20 分钟

## 📚 各模块学习重点和资源

### SQL Medium（核心，60% 时间）

**学习重点**：
- 窗口函数：ROW_NUMBER(), RANK(), DENSE_RANK(), LEAD(), LAG()
- 日期函数：DATE_FORMAT(), DATEDIFF(), DATE_ADD()
- 条件聚合：CASE WHEN + SUM/COUNT
- 组合组件：多个组件的组合使用

**学习方法**：
- 不看答案复写（核心方法）
- 记录语法错误到清单
- 每周复习语法错误清单

**题目来源**：
- 从你已做过的145题中选择 SQL Medium
- LeetCode SQL Medium 新题目扩展

---

### Python Easy（基础，15% 时间）

**学习重点**：
- 哈希表（Dict/Set）：Two Sum, Group Anagrams, Contains Duplicate
- 双指针：Move Zeroes, Remove Duplicates, Valid Palindrome
- 字符串：Valid Anagram, Reverse String
- 基础排序/查找：Binary Search, Top K

**学习方法**：
- 每天 1-2 道 Python Easy，保持基础手感
- 不做新题，复写已做过的题目（如果做过）
- 如果没做过，快速做一遍，理解思路即可

**题目来源**：
- LeetCode Python Easy
- 重点：哈希表、双指针、字符串（这些是 DE 岗位可能考的基础）

**战略定位**：
- 保持基础，不要花太多时间
- 如果面试官考 Python，至少能做出 Easy
- Medium 可以战略性放弃或偶尔尝试

---

### AI Infra（兴趣，15% 时间）

**学习重点**：
- SGLang 架构：调度器、KV Cache、RadixAttention
- 系统设计：LLM 推理系统、请求队列管理、批处理决策
- 核心概念：吞吐量与延迟的权衡、分布式推理、内存管理

**学习方法**：
- 阅读 SGLang 源码和文档
- 阅读系统设计相关文档（A01, A02, B01）
- 理解核心概念，做笔记
- 准备 AI Infra 面试话术

**资源**：
- SGLang 官方文档和源码
- `yc_self_learn/md/` 目录下的 AI 系统设计文档
- A01: LLM 推理系统设计
- A02: 请求队列管理与批处理决策
- B01: LLM 推理系统设计面试问答

**战略定位**：
- 作为兴趣和加分项
- 如果面试 AI Infra 岗位，可以展示对 SGLang 的深入理解
- 如果面试 DE 岗位，可以作为技术敏感度的体现
