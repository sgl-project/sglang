## 🎯 刷题策略

### SQL: 四大支柱（80% 题目都是拼凑）

- **组件 A**: 排名与 Top N（窗口函数）
- **组件 B**: 连续性问题（自连接或偏移）
- **组件 C**: 时间维度的转换
- **组件 D**: 条件聚合（CASE WHEN + SUM）

### Python: 性价比最高的 3 类

- **类别 1**: 双指针/滑动窗口
- **类别 2**: 哈希表（Dict/Set）
- **类别 3**: 基础排序/查找逻辑
