## 📚 各模块学习重点和资源

### SQL Medium（核心，50% 时间）

**学习重点**：
- 窗口函数：ROW_NUMBER(), RANK(), DENSE_RANK(), LEAD(), LAG()
- 日期函数：DATE_FORMAT(), DATEDIFF(), DATE_ADD()
- 条件聚合：CASE WHEN + SUM/COUNT
- 组合组件：多个组件的组合使用

**学习方法**：
- 不看答案复写（核心方法）
- 记录语法错误到清单
- 每周复习语法错误清单

**题目来源**：
- 从你已做过的145题中选择 SQL Medium
- LeetCode SQL Medium 新题目扩展

**四大组件**：
- 组件 A：排名与 Top N（窗口函数）
- 组件 B：连续性问题（LEAD/LAG）
- 组件 C：时间维度转换（DATE_FORMAT）
- 组件 D：条件聚合（CASE WHEN + SUM）

---

### Pandas + Pipeline（重要，25% 时间）

**Pandas 学习重点**：
- 基本操作：读取 CSV、查看、筛选、排序
- 数据清洗：处理缺失值、类型转换、去重
- 数据合并：merge、join
- 数据分组：groupby、聚合函数
- 时间序列：时间处理、日期转换

**Pipeline 学习重点**：
- ETL 概念：Extract（提取）、Transform（转换）、Load（加载）
- Pipeline 设计：数据流动和处理步骤
- 错误处理：数据验证、异常处理
- 性能优化：批量处理、并行处理

**学习方法**：
- 边学边练：看概念 → 写代码 → 做练习
- 实战导向：用实际题目练习（A02、A03、A04）
- Pipeline 思维：每做一个练习都思考"如何改造成 pipeline"

**学习资源**：
- pandas 官方文档的 "10 minutes to pandas"
- 实践项目：用 pandas 重做之前的数据项目
- 代码参考：A02、A03、A04、A05 的代码

**数据库基础**：
- SQLite（最简单，不需要安装服务器）
- MySQL/PostgreSQL（如果需要）
- 用 Python 操作数据库（pandas + SQLAlchemy）

---

### Python Easy（基础，10% 时间）

- 字符串：Valid Anagram, Reverse String
- 基础排序/查找：Binary Search, Top K

- 不做新题，复写已做过的题目（如果做过）
- 如果没做过，快速做一遍，理解思路即可

**战略定位**：

- Medium 可以战略性放弃或偶尔尝试
