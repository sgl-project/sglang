## 方案1：Docker Compose（推荐，最简单）

### 完整配置（开箱即用）

#### docker-compose.yml

```yaml

  sglang-worker-2:
    image: lmsysorg/sglang:latest
    container_name: sglang-worker-2
    restart: unless-stopped
    environment:
      - CUDA_VISIBLE_DEVICES=2
      - HF_TOKEN=${HF_TOKEN}
    command:
      - python3
      - -m
      - sglang.launch_server
      - --model-path
      - ${MODEL_PATH:-/model}
      - --host
      - 0.0.0.0
      - --port
      - "30002"
      - --enable-metrics
    ports:
      - "30002:30000"
    deploy:
      resources:
        reservations:
          devices:
            - driver: nvidia
              device_ids: ["2"]
              capabilities: [gpu]
    healthcheck:
      test: ["CMD-SHELL", "curl -f http://localhost:30000/health || exit 1"]
      interval: 30s
      timeout: 10s
      retries: 3
      start_period: 60s

  # ============================================
  # Traefik (负载均衡，最简单)
  # ============================================
  traefik:
    image: traefik:v2.10
    container_name: traefik
    restart: unless-stopped
    ports:
      - "8080:8080"      # HTTP入口
      - "8081:8081"      # Dashboard
    volumes:
      - /var/run/docker.sock:/var/run/docker.sock:ro
      - ./traefik.yml:/etc/traefik/traefik.yml:ro
      - ./traefik-dynamic.yml:/etc/traefik/dynamic.yml:ro
    command:
      - --api.dashboard=true
      - --api.insecure=true
      - --providers.docker=true
      - --providers.file.directory=/etc/traefik
      - --providers.file.watch=true
    depends_on:
      - sglang-worker-0
      - sglang-worker-1
      - sglang-worker-2

  # ============================================
  # Prometheus (监控指标)
  # ============================================
  prometheus:
    image: prom/prometheus:latest
    container_name: prometheus
    restart: unless-stopped
    ports:
      - "9090:9090"
    volumes:
      - ./prometheus.yml:/etc/prometheus/prometheus.yml:ro
      - prometheus-data:/prometheus
    command:
      - '--config.file=/etc/prometheus/prometheus.yml'
      - '--storage.tsdb.path=/prometheus'
      - '--web.console.libraries=/usr/share/prometheus/console_libraries'
      - '--web.console.templates=/usr/share/prometheus/consoles'

  # ============================================
  # Grafana (监控可视化)
  # ============================================
  grafana:
    image: grafana/grafana:latest
    container_name: grafana
    restart: unless-stopped
    ports:
      - "3000:3000"
    environment:
      - GF_SECURITY_ADMIN_PASSWORD=admin
      - GF_USERS_ALLOW_SIGN_UP=false
    volumes:
      - grafana-data:/var/lib/grafana
      - ./grafana/provisioning:/etc/grafana/provisioning:ro
    depends_on:
      - prometheus

  # ============================================
  # Redis (轻量级队列)
  # ============================================
  redis:
    image: redis:7-alpine
    container_name: redis
    restart: unless-stopped
    ports:
      - "6379:6379"
    volumes:
      - redis-data:/data
    command: redis-server --appendonly yes

volumes:
  prometheus-data:
  grafana-data:
  redis-data:
```

#### traefik.yml（Traefik主配置）

```yaml
# traefik.yml
api:
  dashboard: true
  insecure: true

entryPoints:
  web:
    address: ":8080"

providers:
  docker:
    endpoint: "unix:///var/run/docker.sock"
    exposedByDefault: false
  file:
    filename: /etc/traefik/dynamic.yml
    watch: true
```

#### traefik-dynamic.yml（动态路由配置）

```yaml
# traefik-dynamic.yml
http:
  routers:
    llm-judge:
      rule: "PathPrefix(`/generate`)"
      service: llm-judge-service
      priority: 1
  
  services:
    llm-judge-service:
      weighted:
        services:
          - name: worker-0
            weight: 10
            url: "http://sglang-worker-0:30000"
          - name: worker-1
            weight: 20
            url: "http://sglang-worker-1:30000"
          - name: worker-2
            weight: 15
            url: "http://sglang-worker-2:30000"
  
  middlewares:
    circuit-breaker:
      circuitBreaker:
        expression: "NetworkErrorRatio() > 0.30"
    retry:
      retry:
        attempts: 2
```

#### prometheus.yml（Prometheus配置）

```yaml
# prometheus.yml
global:
  scrape_interval: 10s
  evaluation_interval: 10s

scrape_configs:

          - 'sglang-worker-1:30000'

```

## 方案2：Kubernetes（可选，生产环境）

### 如果必须用K8s，这是最简单的配置

#### sglang-deployment.yaml

```yaml
# sglang-deployment.yaml
apiVersion: apps/v1
kind: Deployment
metadata:
  name: sglang-workers
spec:
  replicas: 3
  selector:
    matchLabels:
      app: sglang-worker
  template:
    metadata:
      labels:
        app: sglang-worker
    spec:
      containers:
        - name: sglang
          image: lmsysorg/sglang:latest
          env:
            - name: HF_TOKEN
              valueFrom:
                secretKeyRef:
                  name: hf-token
                  key: token
          command:
            - python3
            - -m
            - sglang.launch_server
            - --model-path
            - /model
            - --host
            - 0.0.0.0
            - --port
            - "30000"
            - --enable-metrics
          ports:
            - containerPort: 30000
          resources:
            requests:
              nvidia.com/gpu: 1
            limits:
              nvidia.com/gpu: 1
          livenessProbe:
            httpGet:
              path: /health
              port: 30000
            initialDelaySeconds: 60
            periodSeconds: 30
---
apiVersion: v1
kind: Service
metadata:
  name: sglang-service
spec:
  selector:
    app: sglang-worker
  ports:
    - port: 30000
      targetPort: 30000
  type: ClusterIP
```

#### traefik-deployment.yaml

```yaml
# traefik-deployment.yaml

  name: traefik

    app: traefik

    - port: 80
      targetPort: 8080
  type: LoadBalancer
```

## 组件交互说明

### Docker Compose网络（自动创建）

```
┌─────────────────────────────────────────────────┐
│         Docker Network (rad-linter-network)      │
├─────────────────────────────────────────────────┤
│                                                 │
│  ┌──────────────┐    ┌──────────────┐          │
│  │ Traefik      │    │ Prometheus   │          │
│  │ :8080        │    │ :9090        │          │
│  └──────┬───────┘    └──────┬───────┘          │
│         │                   │                   │
│         │ 路由请求           │ 查询指标          │
│         │                   │                   │
│         ▼                   ▼                   │
│  ┌──────────────┐    ┌──────────────┐          │
│  │ Worker-0     │    │ Worker-1     │          │
│  │ :30000       │    │ :30001       │          │
│  └──────────────┘    └──────────────┘          │
│                                                 │
│  ┌──────────────┐    ┌──────────────┐          │
│  │ Worker-2     │    │ Redis        │          │
│  │ :30002       │    │ :6379        │          │
│  └──────────────┘    └──────────────┘          │
│                                                 │
└─────────────────────────────────────────────────┘
```

### 交互流程

#### 1. 请求路由（Traefik → Worker）

```
客户端请求
    ↓
POST http://localhost:8080/generate
    ↓
Traefik (负载均衡)
    ↓ (根据权重选择)
SGLang Worker (GPU推理)
    ↓
返回结果
```

#### 2. 指标采集（Worker → Prometheus）

```
SGLang Worker
    ↓ (暴露/metrics端点)
Prometheus (每10秒抓取)
    ↓ (存储指标)
Grafana (可视化)
```

#### 3. 队列处理（应用 → Redis）

```
应用代码
    ↓ (enqueue)
Redis Queue
    ↓ (dequeue)
Worker处理
```

## 一键启动脚本

### start.sh（Linux/Mac）

```bash
#!/bin/bash

# 一键启动脚本
echo "🚀 Starting Rad-Linter Infrastructure..."

# 检查Docker
if ! command -v docker &> /dev/null; then
    echo "❌ Docker not found. Please install Docker first."
    exit 1
fi

# 检查Docker Compose
if ! command -v docker-compose &> /dev/null; then
    echo "❌ Docker Compose not found. Please install Docker Compose first."
    exit 1
fi

# 检查NVIDIA Docker
if ! docker run --rm --gpus all nvidia/cuda:11.0-base nvidia-smi &> /dev/null; then
    echo "⚠️  NVIDIA Docker not configured. GPU support may not work."
fi

# 检查.env文件
if [ ! -f .env ]; then
    echo "📝 Creating .env file..."
    cat > .env << EOF
HF_TOKEN=your_token_here
MODEL_PATH=/path/to/model
EOF
    echo "⚠️  Please edit .env file with your actual values."
fi

# 创建必要的目录
mkdir -p grafana/provisioning/datasources
mkdir -p grafana/provisioning/dashboards

# 启动服务
echo "📦 Starting services..."
docker-compose up -d

# 等待服务就绪
echo "⏳ Waiting for services to be ready..."
sleep 10

# 检查服务状态
echo "🔍 Checking service status..."
docker-compose ps

echo ""
echo "✅ Services started!"
echo ""
echo "📊 Access points:"
echo "  - Traefik Dashboard: http://localhost:8081"
echo "  - Prometheus: http://localhost:9090"
echo "  - Grafana: http://localhost:3000 (admin/admin)"
echo "  - API Endpoint: http://localhost:8080/generate"
echo ""
echo "📝 To view logs: docker-compose logs -f"
echo "🛑 To stop: docker-compose down"
```

### start.ps1（Windows PowerShell）

```powershell
# start.ps1
Write-Host "🚀 Starting Rad-Linter Infrastructure..." -ForegroundColor Green

# 检查Docker
if (-not (Get-Command docker -ErrorAction SilentlyContinue)) {
    Write-Host "❌ Docker not found. Please install Docker Desktop first." -ForegroundColor Red
    exit 1
}

# 检查Docker Compose
if (-not (Get-Command docker-compose -ErrorAction SilentlyContinue)) {
    Write-Host "❌ Docker Compose not found. Please install Docker Desktop first." -ForegroundColor Red
    exit 1
}

# 检查.env文件
if (-not (Test-Path .env)) {
    Write-Host "📝 Creating .env file..." -ForegroundColor Yellow
    @"
HF_TOKEN=your_token_here
MODEL_PATH=/path/to/model
"@ | Out-File -FilePath .env -Encoding utf8
    Write-Host "⚠️  Please edit .env file with your actual values." -ForegroundColor Yellow
}

# 创建必要的目录
New-Item -ItemType Directory -Force -Path grafana/provisioning/datasources | Out-Null
New-Item -ItemType Directory -Force -Path grafana/provisioning/dashboards | Out-Null

# 启动服务
Write-Host "📦 Starting services..." -ForegroundColor Green
docker-compose up -d

# 等待服务就绪
Write-Host "⏳ Waiting for services to be ready..." -ForegroundColor Yellow
Start-Sleep -Seconds 10

# 检查服务状态
Write-Host "🔍 Checking service status..." -ForegroundColor Green
docker-compose ps

Write-Host ""
Write-Host "✅ Services started!" -ForegroundColor Green
Write-Host ""
Write-Host "📊 Access points:" -ForegroundColor Cyan
Write-Host "  - Traefik Dashboard: http://localhost:8081"
Write-Host "  - Prometheus: http://localhost:9090"
Write-Host "  - Grafana: http://localhost:3000 (admin/admin)"
Write-Host "  - API Endpoint: http://localhost:8080/generate"
Write-Host ""
Write-Host "📝 To view logs: docker-compose logs -f" -ForegroundColor Yellow
Write-Host "🛑 To stop: docker-compose down" -ForegroundColor Yellow
```

## 常见问题

### Q1: GPU不可用？

```bash
# 检查NVIDIA Docker
docker run --rm --gpus all nvidia/cuda:11.0-base nvidia-smi

# 如果失败，安装nvidia-docker2
# Ubuntu: sudo apt-get install nvidia-docker2
# 然后重启Docker: sudo systemctl restart docker
```

### Q2: 端口冲突？

```bash
# 修改docker-compose.yml中的端口映射
# 例如：30000:30000 改为 30010:30000
```

### Q3: 服务启动失败？

```bash
# 查看日志
docker-compose logs -f [service_name]

# 例如
docker-compose logs -f sglang-worker-0
```

### Q4: 如何更新配置？

```bash
# 修改配置文件后，重启服务
docker-compose restart [service_name]

# 或重新加载
docker-compose up -d
```
