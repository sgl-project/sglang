### **ETL Pipeline 学习建议**

#### 1. **理解概念（1 天）**

**ETL 是什么**：
- **Extract（提取）**：从数据源读取数据（CSV、数据库、API）
- **Transform（转换）**：清洗、转换、聚合数据
- **Load（加载）**：保存处理后的数据（数据库、文件、API）

**Pipeline 是什么**：
- 一系列数据处理步骤的**流水线**
- 每个步骤的输出是下一步的输入
- 数据在 pipeline 中**流动**，逐步处理

#### 2. **学习模式（2-3 天）**

**常见 Pipeline 模式**：

```python
# 模式 1：简单 ETL Pipeline
def etl_pipeline():
    # Extract
    raw_data = extract_from_source()
    
    # Transform
    cleaned_data = clean_data(raw_data)
    transformed_data = transform_data(cleaned_data)
    
    # Load
    load_to_destination(transformed_data)

# 模式 2：带验证的 Pipeline
def etl_pipeline_with_validation():
    # Extract
    raw_data = extract_from_source()
    
    # Validate
    if not validate_data(raw_data):
        raise ValueError("Data validation failed")
    
    # Transform
    cleaned_data = clean_data(raw_data)
    transformed_data = transform_data(cleaned_data)
    
    # Validate
    if not validate_data(transformed_data):
        raise ValueError("Transformed data validation failed")
    
    # Load
    load_to_destination(transformed_data)
```

#### 3. **实践项目（3-5 天）**

**项目：设计一个简单的 ETL Pipeline**

```python
# 例子：从多个 CSV 合并数据，清洗后保存到数据库

def extract_data():
    """从多个 CSV 文件提取数据"""
    df1 = pd.read_csv('data1.csv')
    df2 = pd.read_csv('data2.csv')
    return df1, df2

def transform_data(df1, df2):
    """清洗和合并数据"""
    # 清洗
    df1 = df1.dropna()
    df2 = df2.dropna()
    
    # 合并
    merged_df = pd.merge(df1, df2, on='key', how='inner')
    
    # 转换
    merged_df['column'] = pd.to_numeric(merged_df['column'], errors='coerce')
    
    return merged_df

def load_data(df):
    """保存到数据库"""
    from sqlalchemy import create_engine
    engine = create_engine('sqlite:///database.db')
    df.to_sql('table_name', engine, if_exists='replace', index=False)

# Pipeline
def pipeline():
    df1, df2 = extract_data()
    merged_df = transform_data(df1, df2)
    load_data(merged_df)
```

### **数据库学习建议**

#### 1. **从 SQLite 开始（最简单）**

**为什么 SQLite**：
- 不需要安装服务器
- 数据库就是文件
- 语法和标准 SQL 一样

**快速上手（1-2 天）**：

```python
import sqlite3
import pandas as pd

# 连接数据库（如果不存在，会自动创建）
conn = sqlite3.connect('database.db')

# 用 pandas 操作数据库（推荐）
# 读取数据
df = pd.read_sql_query("SELECT * FROM table_name", conn)

# 写入数据
df.to_sql('table_name', conn, if_exists='replace', index=False)

# 关闭连接
conn.close()
```

#### 2. **MySQL/PostgreSQL（可选）**

**如果需要连接 MySQL 或 PostgreSQL**：

```python
from sqlalchemy import create_engine
import pandas as pd

# MySQL
engine = create_engine('mysql+pymysql://user:password@host:port/database')

# PostgreSQL
engine = create_engine('postgresql://user:password@host:port/database')

# 读取数据
df = pd.read_sql_query("SELECT * FROM table_name", engine)

# 写入数据
df.to_sql('table_name', engine, if_exists='replace', index=False)
```

**关键点**：
- 重点是**会用 Python 连接和操作数据库**，不是数据库管理
- 可以从 SQLite 开始（最简单）
