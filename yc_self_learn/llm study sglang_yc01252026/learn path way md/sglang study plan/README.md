## 🚀 快速开始

### 如果你想去RadixArk工作（推荐）

1. **阅读RadixArk计划**: [01_RadixArk工程师路径_8周计划.md](./01_RadixArk工程师路径_8周计划.md)
2. **选择方向**: 从4个方向中选择1个（Inference/Serving、RL/Miles/slime、Kernels/Numerics、Data Infra/Observability）
3. **开始Week 1**: 完成benchmark报告
4. **准备第一个PR**: 根据选择的方向准备
5. **持续贡献**: 保持每周至少1个PR的节奏

### 如果你想加入SGLang RL团队

1. **阅读主文档**: [00_SGLang工程师成长路径.md](./00_SGLang工程师成长路径.md)
2. **完成Warm-up Program**: 按照文档中的指导完成
3. **开始贡献**: 从"good first issue"开始
4. **提交申请**: 发送结果到 rl_team@lmsys.org
