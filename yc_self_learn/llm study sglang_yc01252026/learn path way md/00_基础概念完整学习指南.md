## 📖 第一部分：LLM推理基础

### 1.1 Transformer架构

**概念**: Transformer是LLM的基础架构

**必须理解**:
- Self-Attention机制
- Multi-Head Attention
- Feed-Forward Network (FFN)
- Layer Normalization
- Positional Encoding

**官方文档**:
- [Attention Is All You Need (原始论文)](https://arxiv.org/abs/1706.03762) ⭐ **必读**
- [The Illustrated Transformer](https://jalammar.github.io/illustrated-transformer/) ⭐ **可视化理解**
- [Transformer Architecture Explained](https://huggingface.co/docs/transformers/model_doc/transformer)

**学习时间**: 2-3小时

---

### 1.2 LLM推理流程

**概念**: LLM如何生成token

**必须理解**:
- Prefill阶段（处理prompt）
- Decode阶段（生成token）
- KV Cache的作用
- Token生成循环

**官方文档**:
- [How GPT Works - Andrej Karpathy](https://www.youtube.com/watch?v=wjZofJX0v4M) ⭐ **视频教程**
- [LLM Inference Explained](https://lilianweng.github.io/posts/2023-01-10-inference-optimization/)

**学习时间**: 1-2小时

---

### 1.3 Attention机制详解

**概念**: Attention如何计算

**必须理解**:
- Q, K, V矩阵
- Attention计算公式: `Attention(Q,K,V) = softmax(QK^T/√d_k)V`
- Scaled Dot-Product Attention
- Flash Attention原理

**官方文档**:
- [Flash Attention论文](https://arxiv.org/abs/2205.14135) ⭐ **性能优化必读**
- [Flash Attention官方实现](https://github.com/Dao-AILab/flash-attention)
- [Attention机制详解](https://lilianweng.github.io/posts/2018-06-24-attention/)

**学习时间**: 2-3小时
