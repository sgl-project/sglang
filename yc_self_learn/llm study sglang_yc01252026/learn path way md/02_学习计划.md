# 学习计划 - SGLang Bug分析

## 🎯 学习目标

系统学习理解Issue #17680和#17526所需的所有基础概念，能够：
1. 理解MoE Tensor Parallelism Bug的根本原因
2. 理解GLM Blackwell性能优化的方法
3. 能够阅读和修改SGLang代码
4. 能够实现性能优化方案

## 📅 快速学习路径（2-3周）

### 第1周：基础概念

#### Day 1-2: LLM和Transformer基础
**目标**: 理解LLM的基本原理

**学习内容**:
1. Transformer架构
   - [The Illustrated Transformer](https://jalammar.github.io/illustrated-transformer/) (1小时)
   - [How GPT Works视频](https://www.youtube.com/watch?v=wjZofJX0v4M) (1小时)

**检查点**:

- [ ] 理解Transformer的前向传播
- [ ] 理解LLM如何生成token

#### Day 3-4: GPU和CUDA基础
**目标**: 理解GPU如何工作

**学习内容**:
1. GPU架构基础
   - [An Even Easier Introduction to CUDA](https://developer.nvidia.com/blog/even-easier-introduction-cuda/) (2-3小时)
   - [CUDA C++ Programming Guide](https://docs.nvidia.com/cuda/cuda-c-programming-guide/) (重点章节，4-6小时)

**检查点**:
- [ ] 理解CUDA核心和Tensor Core
- [ ] 理解GPU内存层次结构
- [ ] 理解Kernel、Grid、Block、Thread概念

#### Day 5-7: 量化技术基础
**目标**: 理解量化原理

**学习内容**:
1. 量化基础
   - [PyTorch Quantization](https://pytorch.org/docs/stable/quantization.html) (2-3小时)
   - [FP8 Formats论文](https://arxiv.org/abs/2209.05433) (2-3小时)

**检查点**:
- [ ] 理解FP8和FP4格式
- [ ] 理解量化的目的和原理
- [ ] 理解KV Cache量化
