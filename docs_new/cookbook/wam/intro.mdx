World-Action Models map robot observations, language instructions, and environment state into continuous action chunks. They may use world-model style video/action latent computation internally, but the user-facing workload is action inference for a control loop.

This section documents WAM policies, stateful action rollouts, model-owned cache lifecycle, and evaluation-oriented request schemas.
