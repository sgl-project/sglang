## 1. Model Introduction

[XiaomiMiMo/MiMo-V2.5-Pro](https://huggingface.co/XiaomiMiMo/MiMo-V2.5-Pro) is a Mixture-of-Experts (MoE) language model from XiaomiMiMo with 1.02T total parameters and 42B activated parameters. It uses a hybrid attention architecture and 3-layer Multi-Token Prediction (MTP), with context length up to 1M tokens.

<Note>
MiMo-V2.5-Pro is text-only in the current SGLang support path. Multimodal support for this Pro variant is planned for a future release.
</Note>

**Key features:**

- **Sparse MoE scale**: 1.02T total parameters with 42B activated parameters.
- **Long context**: Supports up to 1M tokens.
- **Hybrid attention**: Uses the MiMo-V2 hybrid attention architecture for long-context efficiency.
- **MTP acceleration**: Supports 3-layer MTP through EAGLE speculative decoding.
- **Reasoning and tool use**: Use the `mimo` reasoning and tool-call parsers for SGLang OpenAI-compatible serving.

## 2. SGLang Installation

Refer to the [official SGLang installation guide](/docs/get-started/install). Use a recent SGLang build that includes MiMo-V2.5-Pro support.

### Configuration tips

- **Platform**: This recipe is for NVIDIA CUDA deployments using FA3 attention and DeepEP.
- **Parallelism**: The example uses `pp=1`, `dp=2`, `ep=16`, and `tp=16`. Adjust `--nnodes`, `--node-rank`, and the parallelism sizes to match your cluster topology.
- **Speculative decoding**: Keep `SGLANG_ENABLE_SPEC_V2=1` when using `--speculative-algorithm EAGLE` with MiMo-V2.5-Pro MTP.
- **Long context**: The command sets `--context-length 1048576`; reduce this value if you need more memory headroom.
- **DeepEP dispatch buffer**: The PR command sets `SGLANG_DEEPEP_NUM_MAX_DISPATCH_TOKENS_PER_RANK=256`. Tune this together with `--max-running-requests` and `--speculative-num-draft-tokens` for high-concurrency serving.

## 4. Model Invocation

Once the server is running, send a text chat request to the OpenAI-compatible endpoint:

```bash Command
curl http://localhost:9001/v1/chat/completions \
  -H "Content-Type: application/json" \
  -d '{
    "model": "XiaomiMiMo/MiMo-V2.5-Pro",
    "messages": [
      {"role": "user", "content": "Explain how Mixture-of-Experts routing improves serving efficiency."}
    ],
    "max_tokens": 512,
    "temperature": 0.7
  }'
```

For general request formats, see [Basic API Usage](/docs/basic_usage/send_request).
